# Notepad | Next.js Application

## Overview

This is a Next.js application created for Tecnical Test for ENSOLVERS.

## Pre-requisites
Is required to have Docker and Docker Compose installed on the host computer.

## Run the project

To run the project isolated open a terminal on /backend directory and then run:
```bash
cp ./.env.example ./.env
docker-compose up
```
